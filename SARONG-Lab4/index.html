Positioning of Elements in CSS

Positioning the elements of an HTML page is no easy feat. CSS provides a set of methods that can be used cumulatively:

  • Margin and spacing: via the margin and padding properties
  • Centering: vertical and horizontal via the text-align and vertical-align properties
  • Display Type: display property
  • Positioning: position property
  • Floating Object: float property

1. Margin and Spacing

Margins and Spacing are used as a priority to position a block with respect to another block or with respect to its parent block. The margin property is used to apply outer spacing to the block and the padding property applies inner spacing to the block.

A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border) wrapped in a parent block (blue border) and vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels.

2. Horizontal and Vertical centering

2.1 Horizontal Centering

To center a text horizontally, the text-align property will be used. To center a block horizontally, one method is to use the automatic margins technique margin:auto - cf. above. Another is to make use of flexible boxes with the justify-content property.

A paragraph of text with a width of 50% that of the parent block, centered horizontally, and whose text is also centered horizontally.

2.2 Vertical Centering

Vertical centering will depend on the container element. If the container block has the display: table-cell property, such as a table for example, you can use vertical-align: middle. Otherwise, you can use the flexible boxes and the align-items property.

Text paragraph 50% wide, centered vertically and horizontally in its parent block.

3. Display Type

Changing the rendering mode of HTML elements via the display property enables easy positioning of the elements relative to one another. Display Block / Inline - Inline-block - Table cell - Flex (space between)

3.1 Display Block / Inline

By default, the main HTML elements use the block (div, p...) or inline (a, span...) renderings. It is possible to modify the rendering mode of an element to change its display mode. Example: use a bullet list containing links to generate a horizontal menu

3.2 Display Inline-Block / Table-cell...

Other types of rendering modes allow, for example, inline and block display to be mixed: inline-block which allows a width, a height, a margins and spacing to be added to an inline block.
Table-cell mode allows table properties to be applied to a block, generally used to center vertically.

3.3 Display Flex

It would take several laboratory exercises to cover all the possibilities presented by flexible boxes. This modern and very powerful rendering mode makes it possible to easily position elements relative to each other.

4. Positioning Type

By default, all elements in the HTML flow are positioned statically => It is possible to position elements by changing the value of the position property

4.2 Relative Position

Relative positioning keeps the element in the flow but allows it to be offset from its initial position via the left, right, top, and bottom properties.

4.3 Absolute Position

Absolute positioning takes the HTML element out of the flow and positions it relative to the wrapping block (having the relative positioning property) according to the values provided by top, left, right and/or bottom.
